## Description

Api of sneakers for practice my knowledge in nest js.

## Installation

```bash
# Execute the comand for running the database in docker
$ docker-compose up -d

# Execute the comand for running api
$ nest start --watch  

# In case of not have install NestJs execute the comand 
$ npm i -g @nestjs/cli

```
